Interface parametrizedFlavor<flavorProps, flavorName>

one of classifications of a type, a complex type in this case, a shape There are two kinds of flavors, complex and elementary

  • Complex flavored type is generated by flavor factory,e.g. shape factory
  • elementary flavored types are more fundamental,e.g. string or number
